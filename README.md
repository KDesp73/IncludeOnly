# IncludeOnly

Is a repo hosting header-only C utility libraries

## Fetch

Get all libraries locally

```sh
bash <(curl -s https://raw.githubusercontent.com/KDesp73/IncludeOnly/refs/heads/main/scripts/installer.sh)
```
